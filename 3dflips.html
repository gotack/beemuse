Usage

The relatively-positioned .flipper element requires a height to be specified, which should include in addition a bottom margin of your choosing. Additionally a width should be specified, to avoid the flip trigger area from auto-expanding in its containing element. Keep in mind how setting these may affect responsive versions of .flipper elements (the width and height may need to be reset at the appropriate breakpoints).

Degrades gracefully (simply shows back or front without animation). IE10 support.

Credit: homebrewed as most demos use jQuery or don't get it quite right :)

HTML

<div class="flipper">
  <a href="#">
    <div class="flip front">
      ...
    </div>
    <div class="flip back">
      ...
    </div>
  </a>
</div>

SCSS

.flipper {
  position: relative;
  @include perspective;
  .flip {
    position: absolute;
    top: 0;
    left: 0;
    @include transform;
    @include backface;
    @include transition(0.75s);
  }
  .front {
    z-index: 5;
  }
  .back {
    z-index: 0;
    @include rotateY(-180deg);
  }
  &:hover .front {
    z-index: 5;
    @include rotateY(180deg);
  }
  &:hover .back {
    z-index: 10;
    @include rotateY(0deg);
  }
}

SCSS Mixins

@mixin transition($transition) {
  -webkit-transition: $transition;
     -moz-transition: $transition;
      -ms-transition: $transition;
       -o-transition: $transition;
          transition: $transition;
}
@mixin rotateY($degrees) {
  -webkit-transform: rotateY($degrees);
     -moz-transform: rotateY($degrees);
      -ms-transform: rotateY($degrees);
       -o-transform: rotateY($degrees);
          transform: rotateY($degrees);
}
@mixin perspective($degree: 1000) {
  -webkit-perspective: $degree;
     -moz-perspective: $degree;
      -ms-perspective: $degree;
       -o-perspective: $degree;
         -perspective: $degree;
}
@mixin transform($style: preserve-3d) {
  -webkit-transform-style: $style;
     -moz-transform-style: $style;
      -ms-transform-style: $style;
       -o-transform-style: $style;
         -transform-style: $style;
}
@mixin backface($visibility: hidden) {
  -webkit-backface-visibility: $visibility;
     -moz-backface-visibility: $visibility;
      -ms-backface-visibility: $visibility;
       -o-backface-visibility: $visibility;
         -backface-visibility: $visibility;
}
